## Acolyte

Acolyte is the lightweight terminal file manager. Written in C++, it provides best responsibility when managing files from terminal

## Installation

Just download .zip archive from GitHup and run ./acolyte in terminal:

```
./acolyte
```

## Usage

For better understanding of functionality, run help in Acolyte's terminal window to get its docs:

```
> help
```
